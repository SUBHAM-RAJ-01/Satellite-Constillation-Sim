================================================================================
                    GETTING STARTED - FIRST TIME USERS
================================================================================

Welcome! This guide will get you running in 2 minutes.


STEP 1: VERIFY PYTHON
----------------------
Open your terminal/command prompt and type:

    python --version

You should see: Python 3.6 or higher

If not, download Python from: https://www.python.org/downloads/


STEP 2: RUN THE SIMULATION
---------------------------
In your terminal, type:

    python main.py

You'll see a menu like this:

    Select Simulation Mode:
      1. Full Simulation (OSPF routing + Partition comparison)
      2. Full Simulation (TSA routing + Partition comparison)
      3. Compare Both Routing Protocols
      4. Performance Model Only (Theoretical)
      5. Exit


STEP 3: CHOOSE AN OPTION
-------------------------
For your first run, we recommend option 4 (fastest):

    Enter your choice (1-5): 4

This runs in less than 1 second and shows the core results.


WHAT YOU'LL SEE
---------------
The simulation will display:

    1. PROJECT OVERVIEW
       - 900 satellites, 1500 users, 20 containers
       - Total CPU work, memory usage, serial time

    2. LOAD DISTRIBUTION COMPARISON
       - UTP: 30% load imbalance
       - LBTP: 5% load imbalance

    3. FINAL SPEEDUP ANALYSIS
       - UTP Speedup: ~15.4x
       - LBTP Speedup: ~19.05x
       - LBTP is 24% better!


NEXT STEPS
----------
After your first run:

1. Try option 1 for full simulation (takes 10-30 seconds)
2. Try option 3 to compare routing protocols
3. Run: python demo.py to see component demos
4. Read QUICKSTART.md for more details

UNDERSTANDING THE RESULTS
--------------------------

KEY METRIC: Speedup
  - Shows how much faster parallel execution is vs serial
  - UTP (naive): ~15.4x speedup
  - LBTP (optimized): ~19.05x speedup
  - Higher is better!

KEY METRIC: Load Imbalance
  - Shows how evenly work is distributed
  - UTP: ~30% imbalance (uneven)
  - LBTP: ~5% imbalance (balanced)
  - Lower is better!

THE RESULT:
  LBTP's intelligent load balancing achieves 24% better performance!
